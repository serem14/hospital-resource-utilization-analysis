# Hospital Resource Utilization Analysis

## Project Overview

This project analyzes hospital patient data to identify patterns in healthcare resource utilization. The goal is to support data-driven decision-making in patient management and resource allocation.

## Objectives

- Identify high-cost patients  
- Analyze hospital visit patterns  
- Segment patients based on utilization  
- Provide actionable healthcare recommendations  

## Tools Used

- SQL (Data querying and analysis)  
- Excel (Data preparation and segmentation)  
- Power BI (Data visualization and dashboarding)  

## Dataset Description

The project uses two datasets:

### Patients Table
- patient_id  
- age  
- gender  
- age_group  

### Visits Table
- visit_id  
- patient_id  
- visits  
- cost  
- cost_intensity  
- burden  

## Dataset Source

The dataset used in this project is a simulated/structured healthcare dataset created for analytical practice.

## Key Metrics

- Total hospital cost analyzed  
- Patient count segmented by utilization  
- High-burden patient percentage  
- Average cost per visit  

## SQL Analysis

Key analyses performed:

- Patients with high visit frequency  
- Cost distribution across age groups  
- High-burden patient identification  
- Patients with above-average total cost (advanced subquery)  

## Dashboard (Power BI)

The dashboard presents:

- Total hospital cost  
- Visits by age group  
- Cost distribution by gender  
- High-burden patient segmentation  

## Key Insights

- Senior patients showed the highest hospital utilization  
- A small percentage of patients contributed to a large portion of total cost  
- High-burden patients require targeted intervention  
- Frequent low-cost visits indicate recurring manageable conditions  

## Recommendations

- Implement follow-up programs for high-risk patients  
- Introduce preventive care for senior populations  
- Optimize resource allocation toward high-cost patients  
- Enhance patient education to reduce recurrent visits  

## Conclusion

This analysis demonstrates how data analytics can support efficient healthcare management through patient segmentation and cost analysis.
